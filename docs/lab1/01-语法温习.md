# 语法温习

在开始写贪吃蛇之前，快速回顾一下本实验会用到 C++ 语法。如果你对某个知识点感到陌生，建议先翻翻课本或问问 LLM。

---

## 变量与数据类型

```cpp
int   score = 0;       // 整数
bool  running = true;  // 布尔值，只有 true / false
```

### 枚举 `enum`

当你需要一组有名字的整数常量时，用枚举会很方便：

```cpp
enum Direction { UP, DOWN, LEFT, RIGHT };
Direction dir = RIGHT;   // dir 现在等于 3

// 配合 switch 使用
switch (dir) {
    case UP:    /* ... */ break;
    case DOWN:  /* ... */ break;
    case LEFT:  /* ... */ break;
    case RIGHT: /* ... */ break;
}
```

> 枚举本质上就是整数，但比直接写 `0, 1, 2, 3` 可读性好得多。

### 结构体 `struct`

把多个相关数据打包成一个新类型：

```cpp
struct Point {
    int x;
    int y;
};

Point p;
p.x = 10;
p.y = 5;
```

在游戏中，每条蛇的身体节点就是一个 `Point`，包含 `x` 和 `y` 坐标。

---

## 数组

```cpp
Point snake[600];   // 600 个 Point，蛇身最大长度
int arr[100];       // 100 个 int
```

- 下标从 `0` 开始，`snake[0]` 是第一个元素（蛇头）
- 遍历用 `for` 循环：`for (int i = 0; i < len; i++)`
- **绝对不能越界**——访问 `snake[-1]` 或 `snake[600]`（长度 600 时）会导致程序崩溃

## 控制流

### if / else if / else

```cpp
if (score >= 100) {
    // ...
} else if (score >= 60) {
    // ...
} else {
    // ...
}
```

### switch

```cpp
switch (key) {
    case 'W': case 'w': /* 上 */ break;
    case 'S': case 's': /* 下 */ break;
    default: /* 其他情况 */ break;
}
```

一个常见的坑：忘了写 `break` 会"穿透"到下一个 case，导致意想不到的结果。

### while 循环

```cpp
while (running) {
    // 只要 running 是 true，就一直重复执行
}
```

游戏主循环就是一个 `while`——每一帧跑一遍，直到游戏结束。

### for 循环

```cpp
// 正向：0 → 1 → 2 → ... → n-1
for (int i = 0; i < n; i++) { /* ... */ }

// 反向：n-1 → n-2 → ... → 0
for (int i = n - 1; i >= 0; i--) { /* ... */ }
```

### do-while 循环

和 while 的区别在于**至少执行一次**：

```cpp
do {
    // 生成随机坐标
} while (/* 坐标和蛇身重叠 */);  // 重叠就重新来
```

## 函数

```cpp
// 无返回值
void drawSnake() { /* ... */ }

// 返回 bool
bool checkCollision() {
    if (/* 碰到了 */) return true;
    return false;
}
```

把功能拆成小函数是很好的习惯——每个函数只做一件事，代码读起来清晰，调试也方便。

---

## 随机数

```cpp
#include <cstdlib>   // rand(), srand()
#include <ctime>     // time()

srand((unsigned int)time(nullptr));  // 程序启动时初始化一次

int x = rand() % 50 + 1;   // [1, 50] 范围的整数
```

- `rand() % N` 的范围是 `[0, N-1]`
- 加 1 后变成 `[1, N]`
- 不调用 `srand` 的话，每次运行 `rand()` 生成的序列都是一样的

## 逻辑运算符

```cpp
if (x < 1 || x > 50)      // 或（or）：超出边界
if (a == b && c == d)     // 且（and）：两个条件都满足
if (!running)             // 非（not）：running 为 false
```

---

## 取模 `%`

```cpp
score % 50 == 0   // 判断 score 能否被 50 整除
10 % 3   // → 1（余数）
15 % 5   // → 0（整除）
```

游戏中用它来判断"每得 50 分就加速一次"。
