# 语法学习

本实验会用到三个核心知识点：

- `vector`：动态数组，用来存储数据和 shape
- `template`：模板，让 `NdArray` 支持 `int`、`double` 等多种类型
- `operator()`：运算符重载，实现 `a({i, j})` 多维下标访问

建议先读完这一份，再去看 `student` 文件夹里的代码。

## 1. vector —— 动态数组

`vector<T>` 是 C++ 标准库里的动态数组，可以自动管理内存、随时增减元素。

```cpp
#include <vector>
using namespace std;

vector<int> v;           // 空的 int 数组
vector<double> d(10);    // 10 个 double，初始值 0.0
vector<int> shape = {2, 3, 4};  // 初始化列表
```

### 常用操作

```cpp
v.size()         // 返回元素个数
v.resize(n)      // 重新调整大小
v[i]             // 下标访问（不会检查越界）
v.push_back(x)   // 在末尾添加元素
```

### 遍历 vector

```cpp
// 方式 1: 下标
for (int i = 0; i < (int)v.size(); i++) {
    cout << v[i] << endl;
}

// 方式 2: 范围 for（本实验最常用）
for (int x : v) {
    cout << x << endl;
}

// 方式 3: 遍历 shape 累乘求总元素数
int total = 1;
for (int s : shape) {
    total *= s;
}
```

### 本实验中的用法

```cpp
vector<T> data_;       // 存储数组元素（一维连续内存）
vector<int> shape_;    // 存储各维度大小，如 {2, 3}
vector<int> strides_;  // 存储各维度步长，如 {3, 1}
```

## 2. template —— 模板

模板让你写一份代码，就能支持多种类型。

### 函数模板

```cpp
// 一个能比较任意类型的 max 函数
template<typename T>
T myMax(T a, T b) {
    return a > b ? a : b;
}

// 使用
int x = myMax(3, 5);           // T = int
double y = myMax(3.14, 2.71);  // T = double
```

### 类模板

```cpp
template<typename T>
class Box {
private:
    T value_;
public:
    Box(T v) : value_(v) { }
    T get() const { return value_; }
};

// 使用
Box<int> intBox(42);
Box<string> strBox("hello");
```

### 本实验中的用法

```cpp
template<typename T>
class NdArray { ... };

// 使用时需要指定类型：
NdArray<int>    a({2, 3});   // int 类型的二维数组
NdArray<double> b({3, 4});   // double 类型的二维数组
```

### 重要：模板类的实现必须放在头文件

普通类可以 `.h` 声明 + `.cpp` 实现。但模板类的代码**必须全部放在 `.h`**，因为编译器在见到 `NdArray<int>` 时才根据模板生成真正的代码。如果实现在 `.cpp` 里，编译器看不到，就会报 undefined reference。

## 3. operator() —— 函数调用运算符

C++ 可以重载 `operator()`，让对象像函数一样被调用。

### 基本语法

```cpp
class Adder {
public:
    int operator()(int a, int b) const {
        return a + b;
    }
};

Adder add;
int result = add(3, 4);  // 看起来像函数调用，实际是 add.operator()(3, 4)
```

### 本实验中的用法

用 `operator()` 实现多维下标访问：

```cpp
T& operator()(const vector<int>& indices) {
    int offset = 0;
    for (int i = 0; i < ndim(); i++) {
        offset += indices[i] * strides_[i];
    }
    return data_[offset];
}
```

调用方式：

```cpp
NdArray<int> a({2, 3});
a({0, 1}) = 42;       // 等价于 a.operator()({0, 1})
cout << a({0, 1});    // 输出 42
```

`{0, 1}` 会隐式构造一个 `vector<int>`，传给 `operator()`。

### 需要两个版本

```cpp
T& operator()(const vector<int>& indices);             // 可读写
const T& operator()(const vector<int>& indices) const;  // 只读
```

- 非 const 版本：用于赋值 `a({0,1}) = 42`
- const 版本：用于读取 `cout << a({0,1})`

两个版本的 offset 计算逻辑完全一样。

## 4. static 成员函数

`static` 成员函数不属于某个具体对象，而是属于整个类。调用时用 `类名::函数名`。

```cpp
template<typename T>
class NdArray {
public:
    static NdArray zeros(const vector<int>& shape) {
        NdArray arr(shape);
        // 把所有元素设为 0
        return arr;
    }
};

// 调用：不需要先创建对象
auto z = NdArray<double>::zeros({3, 4});
```

本实验中 `zeros`、`ones`、`full`、`arange` 都是 static 函数。它们的共同模式是：

1. 先构造一个数组
2. 填充数据
3. 返回

## 5. const 成员函数

函数后面加 `const` 表示"这个函数不会修改对象"：

```cpp
vector<int> shape() const { return shape_; }   // 只读，加 const
int size() const { ... }                        // 只读，加 const
void print() const { ... }                      // 只读，加 const
```

但 `operator()` 的非 const 版本返回的是可修改的引用，会修改对象，所以不能加 const：

```cpp
T& operator()(const vector<int>& indices) { ... }  // 可修改，不加 const
```

## 6. 本实验涉及的 C++11 特性

- **`auto`**：自动推导类型
  ```cpp
  auto z = NdArray<double>::zeros({3, 4});  // 省去写 NdArray<double>
  ```

- **range-for**：遍历容器
  ```cpp
  for (int s : shape) total *= s;
  ```

- **`initializer_list`（隐式）**：`{2, 3}` 可以直接构造成 `vector<int>`
  ```cpp
  NdArray<int> a({2, 3});  // {2, 3} → vector<int>{2, 3}
  ```
