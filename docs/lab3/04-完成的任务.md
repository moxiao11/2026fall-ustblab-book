## TODO 1：构造函数

```cpp
NdArray(const vector<int>& shape)
```

要求：

1. 把 `shape` 存到 `shape_`
2. 计算 `strides_`（row-major，行优先）：
   - 最后一维步长 = 1
   - 从后往前：`strides_[i] = strides_[i+1] * shape[i+1]`
3. 计算总元素数 `total = shape[0] * shape[1] * ...`
4. 用 `data_.resize(total)` 分配空间

示例：

```text
shape = {2, 3, 4}

strides[2] = 1
strides[1] = 1 * 4 = 4
strides[0] = 4 * 3 = 12

strides = {12, 4, 1}
total = 2 * 3 * 4 = 24
```

## TODO 2：shape / size / ndim

三个访问器：

| 方法 | 返回什么 | 示例（shape={2,3}） |
|---|---|---|
| `shape()` | `shape_` 的副本 | `{2, 3}` |
| `size()` | 所有维度的乘积 | `6` |
| `ndim()` | shape 的长度 | `2` |

要求：

- `size()` 遍历 shape 累乘，初始值用 1
- `ndim()` 返回 `(int)shape_.size()`

## TODO 3：下标访问 `operator()`

```cpp
T& operator()(const vector<int>& indices);              // 非 const：可修改
const T& operator()(const vector<int>& indices) const;  // const：只读
```

公式：

```
offset = indices[0] * strides_[0]
       + indices[1] * strides_[1]
       + ...
```

然后返回 `data_[offset]`。

要求：

- 两个版本的 offset 计算逻辑**完全一样**
- 非 const 版返回 `T&`（可赋值），const 版返回 `const T&`（只读）

示例：

```cpp
NdArray<int> a({2, 3});
a({0, 1}) = 42;       // offset = 0*3 + 1*1 = 1, data_[1] = 42
cout << a({0, 1});    // 输出 42
```

## TODO 4：zeros

```cpp
static NdArray zeros(const vector<int>& shape);
```

要求：

1. 用 `shape` 构造一个 `NdArray arr`
2. 遍历 `arr.size()` 次，把每个元素设为 `T(0)`
3. 返回 `arr`

示例：

```cpp
auto z = NdArray<double>::zeros({3, 4});  // 3x4 全零矩阵
```

## TODO 6：arange

```cpp
static NdArray arange(T start, T end, T step);
```

创建一维等差数组，从 `start` 到 `end`（不含），步长 `step`。

思路：

1. 用临时 `vector<T> values` 收集值
2. `for (T v = start; v < end; v += step)` 逐个 `push_back`
3. 用 `values` 的长度创建一维 `NdArray`
4. 把 values 的值拷贝到 `data_` 中

示例：

```cpp
auto r = NdArray<int>::arange(0, 10, 2);
// → [0, 2, 4, 6, 8]

auto d = NdArray<double>::arange(0.0, 1.0, 0.2);
// → [0, 0.2, 0.4, 0.6, 0.8]
```

## TODO 7：full

```cpp
static NdArray full(const vector<int>& shape, T value);
```

创建形状为 `shape`、所有元素为 `value` 的数组。

和 `zeros`/`ones` 逻辑完全一样，只是把值设为传入的 `value` 参数。

示例：

```cpp
auto f = NdArray<double>::full({2, 3}, 3.14);  // 2x3 全是 3.14
auto f2 = NdArray<int>::full({4}, 7);           // 一维 [7, 7, 7, 7]
```

## TODO 8：fill

```cpp
void fill(T value);
```

把当前数组的所有元素填充为 `value`。遍历 `data_`，逐一赋值。

示例：

```cpp
NdArray<int> a({2, 3});
a.fill(42);   // 所有元素变成 42
a.fill(-1);   // 所有元素变成 -1
```

---

## TODO 9：reshape

```cpp
NdArray reshape(const vector<int>& new_shape) const;
```

返回一个**新数组**，数据不变但形状改变。要求新形状的元素总数与原数组一致。

思路：

1. 用 `new_shape` 构造新 `NdArray`
2. 将原数组的 `data_` 按顺序拷贝到新数组的 `data_`

示例：

```cpp
NdArray<int> a({2, 3});  // 2x3
// ... 填入 0,1,2,3,4,5
auto r = a.reshape({3, 2});  // 变成 3x2，数据不变
auto r2 = a.reshape({6});    // 变成一维 [0,1,2,3,4,5]
```

## 常见坑

- **strides 算反了？** row-major 是"最后一维步长 = 1"，循环从 `ndim - 1` 往 `0` 走
- **size() 返回值不对？** 初始化为 1，遍历 shape 累乘
- **arange 的 step 是 double 时数量不对？** 用临时 vector 收集再取 `.size()`，不要预先算数量（浮点精度问题）
- **operator() 两个版本逻辑一样？** 是的，const 只是返回类型不同（`const T&`），offset 计算完全一样
- **zeros/ones 忘记遍历了？** 构造完后需要 for 循环给每个元素赋值
- **编译报 undefined reference？** 检查是否把所有 `.cpp` 一起编译：`g++ -std=c++11 main.cpp test.cpp -o main.exe`
- **单独编译 NdArray.h 报错？** 头文件不能单独编译，它是被 `.cpp` include 的
- **赋值没反应？** 检查 `operator()` 非 const 版本是否返回了 `T&`（引用），而不是 `T`（拷贝）
