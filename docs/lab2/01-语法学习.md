## 1. 随机数

C++ 里可以用 `rand()` 生成随机整数。

```cpp
#include <cstdlib>
#include <ctime>

srand((unsigned int)time(nullptr));

int x = rand();
```

`rand()` 返回一个比较大的整数。如果想得到某个范围内的数字，可以用取模 `%`。

例如随机得到 `0 ~ 7`：

```cpp
int index = rand() % 8;
```

本实验里有一个句子数组：

```cpp
const int SENTENCE_COUNT = 8;

const string SENTENCES[SENTENCE_COUNT] = {
    "Practice makes progress.",
    "C plus plus is powerful and fast.",
    "Debugging is part of programming.",
    "Small steps lead to big results.",
    "Read the code before changing it.",
    "A clear idea makes clean code.",
    "Typing accuracy matters more than speed.",
    "Functions help split a big task into small tasks."
};
```

所以随机选一句可以写成：

```cpp
int index = rand() % SENTENCE_COUNT;
string target = SENTENCES[index];
```

注意：`srand((unsigned int)time(nullptr));` 通常只在程序开始时调用一次，放在 `main()` 里就可以。

## 2. 计时功能

本实验使用 C++ 标准库里的 `<chrono>` 计时。

```cpp
#include <chrono>
```

记录开始时间：

```cpp
auto start = chrono::steady_clock::now();
```

记录结束时间：

```cpp
auto finish = chrono::steady_clock::now();
```

计算经过了多少秒：

```cpp
chrono::duration<double> elapsed = finish - start;
double seconds = elapsed.count();
```

`steady_clock` 适合用来测量一段代码运行了多久。它不会因为系统时间被手动修改而突然跳变。

在本实验中，我们希望计时从用户开始输入前开始，到用户按下 Enter 后结束：

```cpp
console.startTimer();
string typed = console.readTypedLine();
double seconds = console.stopTimer();
```

## 3. 类与对象复习

类可以把一组相关的数据和函数放在一起。

例如本实验里的 `TypingConsole` 类负责：

- 显示标题
- 显示输入提示
- 读取用户输入
- 开始计时
- 停止计时
- 显示结果
- 询问是否再玩一次

类的声明放在 `.h` 文件里：

```cpp
class TypingConsole {
public:
    void showHeader() const;
    void startTimer();
    double stopTimer() const;

private:
    chrono::steady_clock::time_point startTime;
};
```

其中：

- `public`：外部代码可以调用
- `private`：只有类内部可以访问
- `startTime`：保存开始计时的时间点

创建对象：

```cpp
TypingConsole console;
```

调用成员函数：

```cpp
console.showHeader();
console.startTimer();
```

## 4. const 成员函数

你会看到一些函数后面有 `const`：

```cpp
void showHeader() const;
double stopTimer() const;
```

这表示这个成员函数不会修改对象内部的数据。

比如 `showHeader()` 只是打印标题，不需要改变对象状态，所以可以写成 `const`。

但 `startTimer()` 会修改 `startTime`，所以它不能写成 `const`：

```cpp
void startTimer();
```

## 5. 结构体 TypingStats

本实验用 `TypingStats` 保存一次打字测试的结果：

```cpp
struct TypingStats {
    double seconds;
    int correctChars;
    int errors;
    double wpm;
    double accuracy;
};
```

这样 `showResult()` 不需要传一长串参数，只要传一个结构体：

```cpp
TypingStats stats = buildStats(target, typed, seconds);
console.showResult(stats);
```

这就是“把相关数据打包”的好处。

## 6. 本实验的两个公式

WPM 是 words per minute，每分钟单词数。打字测速里常用规则是：

```text
1 word = 5 characters
WPM = correctChars / 5.0 / minutes
```

准确率：

```text
accuracy = correctChars * 100.0 / typedChars
```

注意：

- 如果 `minutes <= 0`，WPM 应该返回 `0`
- 如果 `typedChars == 0`，准确率应该返回 `0`
- 用 `5.0` 而不是 `5`，可以避免整数除法
