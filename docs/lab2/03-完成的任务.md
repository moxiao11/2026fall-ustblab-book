### TODO 2：`countCorrectChars()`

统计用户输入中有多少字符是正确的。

要求：

- 将 `target` 和 `typed` 逐字符比较
- 只比较两个字符串重叠的部分
- 同一位置字符相同，正确字符数加 1
- 不要越界访问字符串

例子：

```text
target: hello
typed:  hxllo
```

第 1、3、4、5 个字符正确，所以正确字符数是 `4`。

---

### TODO 3：`countErrors()`

统计错误数。

要求：

- 重叠部分中，字符不同算错误
- 用户少打的字符也算错误
- 用户多打的字符也算错误

例子：

```text
target: hello
typed:  he
```

少打了 `llo`，错误数是 `3`。

再比如：

```text
target: hello
typed:  hellooo
```

多打了 `oo`，错误数是 `2`。

---

### TODO 4：`calculateWPM()`

计算打字速度 WPM。

要求：

- WPM 表示 words per minute
- 本实验使用常见规则：`1 word = 5 correct characters`
- 公式：

```text
WPM = correctChars / 5.0 / minutes
```

注意：

- 如果 `minutes <= 0`，返回 `0`
- 要写 `5.0`，不要写 `5`，避免整数除法

---

### TODO 5：`calculateAccuracy()`

计算准确率。

要求：

- 公式：

```text
accuracy = correctChars * 100.0 / typedChars
```

注意：

- 如果 `typedChars == 0`，返回 `0`
- 返回值是百分比，例如 `95.5` 表示 `95.5%`

---

### TODO 6：`buildStats()`

把一次测试的所有结果装进 `TypingStats` 结构体。

要求：

- 创建一个 `TypingStats stats`
- 设置 `stats.seconds = seconds`
- 调用 `countCorrectChars()` 得到正确字符数
- 调用 `countErrors()` 得到错误数
- 用 `seconds / 60.0` 得到分钟数
- 调用 `calculateWPM()` 得到 WPM
- 调用 `calculateAccuracy()` 得到准确率
- 最后 `return stats`

---

### TODO 9：`TypingConsole::stopTimer()`

停止计时，并返回经过的秒数。

要求：

- 获取当前时间
- 用当前时间减去 `startTime`
- 把时间差转换成 `chrono::duration<double>`
- 返回 `elapsed.count()`

示例结构：

```cpp
chrono::steady_clock::time_point finishTime = chrono::steady_clock::now();
chrono::duration<double> elapsed = finishTime - startTime;
return elapsed.count();
```

## 常见坑

- **输入读不到空格？** 用 `getline(cin, typed)`，不要用 `cin >> typed`。
- **WPM 结果一直是 0？** 检查有没有用 `5.0`，不要写成纯整数除法。
- **编译时报 undefined reference？** 多半是只编译了 `main.cpp`，需要把所有 `.cpp` 一起编译。
- **`-o` 报错找不到 exe？** `-o` 后面要紧跟输出文件名，例如 `g++ -std=c++11 *.cpp -o main.exe`。
- **类成员函数编译不过？** 在 `.cpp` 里实现成员函数时要写 `TypingConsole::`。
- **准确率超过 100？** 正确字符数只应该统计到两个字符串的较短长度。
