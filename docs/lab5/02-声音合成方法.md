# 声音合成方法 — 频率、正弦波、谐波与 ADSR

## 一、背景知识

本实验的核心思路是：**把一首歌拆成多个音符对象，再把每个音符合成为一段音频采样，最后拼接成完整歌曲。**

程序中主要包含三层逻辑：

- **乐谱层**：使用 `Song` 和 `Note` 表示歌曲和音符
- **合成层**：使用 `FrequencyTable`、`ADSREnvelope`、`AdditiveSynthesizer` 生成声音采样
- **输出层**：使用 `WavWriter` 保存音频文件，使用 `MusicPlayer` 播放音频

### 1.1 音符频率

音符可以用频率表示，例如：

| 音符 | 频率 |
|------|------|
| C4 | 261.63 Hz |
| D4 | 293.66 Hz |
| E4 | 329.63 Hz |
| F4 | 349.23 Hz |
| G4 | 392.00 Hz |
| A4 | 440.00 Hz |
| B4 | 493.88 Hz |

本实验使用十二平均律公式计算频率：

```cpp
frequency = 440.0 * pow(2.0, (midiNumber - 69) / 12.0);
```

其中 A4 的 MIDI 编号是 69，频率是 440 Hz。

### 1.2 正弦波合成

一个最简单的音符可以看成一段正弦波：

```cpp
sample = amplitude * sin(2 * pi * frequency * t);
```

- `frequency`：音符频率
- `t`：当前采样点对应的时间
- `amplitude`：音量大小

### 1.3 谐波叠加

单一正弦波声音比较单薄，类似电子蜂鸣器。为了让声音更丰富，本实验使用多个频率叠加：

```cpp
sample =
    1.00 * sin(2 * pi * f * t) +
    0.45 * sin(2 * pi * 2 * f * t) +
    0.22 * sin(2 * pi * 3 * f * t) +
    0.10 * sin(2 * pi * 4 * f * t);
```

这就是简单的**加法合成**。每个谐波的频率是基频的整数倍，振幅依次递减。

### 1.4 ADSR 包络

ADSR 用于控制一个音符从开始到结束的音量变化：

| 阶段 | 含义 |
|------|------|
| Attack | 起音，音量从 0 增大到最大 |
| Decay | 衰减，音量从最大下降到稳定音量 |
| Sustain | 保持，音量保持在稳定值 |
| Release | 释音，音量逐渐下降到 0 |

没有包络时，音符开始和结束会比较突兀；加入 ADSR 后声音会更自然。

## 八、常见坑

- **音量太大导致爆音**

  可以调小 `SynthConfig` 中的 `amplitude`，例如改成 `0.30`。

- **音符结尾有明显咔哒声**

  可以适当增大 `release`，让音量逐渐衰减到 0。
