# 实验任务清单

本实验略难。要求部分甚至会告诉你使用哪个变量，减轻部分难度，但仍旧不易。

## 一、实验目标

在本实验中，你将使用 C++ 实现一个**简易音乐合成器**。程序不使用外部音频素材，而是根据音符、节拍、频率和合成算法生成声音采样，保存为 `.wav` 文件，并调用系统接口进行播放。

通过本实验你将掌握：

1. 类与对象的基本设计方法
2. 音符频率、采样率、采样点等数字音频基本概念
3. 正弦波合成和谐波叠加的基本思想
4. ADSR 包络对声音音量变化的控制
5. WAV 文件的基本写入流程

## 二、程序架构

程序中主要包含三层逻辑：

- **乐谱层**：使用 `Song` 和 `Note` 表示歌曲和音符
- **合成层**：使用 `FrequencyTable`、`ADSREnvelope`、`AdditiveSynthesizer` 生成声音采样
- **输出层**：使用 `WavWriter` 保存音频文件，使用 `MusicPlayer` 播放音频

| 类名 | 职责 |
|------|------|
| `Note` | 保存单个音符的音名和节拍 |
| `Song` | 保存歌曲标题和音符序列 |
| `SynthConfig` | 保存速度、音量、ADSR 参数 |
| `FrequencyTable` | 将 `"C4"`、`"A4"` 等音名转换为频率 |
| `ADSREnvelope` | 根据时间计算当前音量包络值 |
| `AdditiveSynthesizer` | 根据乐谱生成 PCM 采样数据 |
| `WavWriter` | 将采样数据写入 WAV 文件 |
| `MusicPlayer` | 播放生成的 WAV 文件 |

### TODO 1 — FrequencyTable::getFrequency

程序需要把 `"C4"` 拆成音名 `C` 和八度 `4`，再通过十二平均律公式计算频率。

要求：

1. `"REST"` 返回 `0.0`
2. 用 `substr()` 取音名
3. 用 `back()` 取八度
4. 用 `semitoneMap.find()` 查找半音编号
5. 用公式 `440.0 * pow(2.0, (midiNumber - 69) / 12.0)` 计算频率

### TODO 2 — ADSREnvelope::getValue

完成 Attack、Decay、Sustain、Release 四个阶段的音量计算。

要求：

1. Attack：音量从 `0` 线性增加到 `1`
2. Decay：音量从 `1` 线性下降到 `sustainLevel`
3. Sustain：返回 `sustainLevel`
4. Release：音量从 `sustainLevel` 线性下降到 `0`
5. 使用 `std::min` 限制各阶段时长

### TODO 3 — AdditiveSynthesizer::synthSample

完成一个采样点的谐波叠加。

要求叠加：

```cpp
1.00 * sin(2*pi*f*t)
0.45 * sin(2*pi*2*f*t)
0.22 * sin(2*pi*3*f*t)
0.10 * sin(2*pi*4*f*t)
```

最后除以 `1.77`，避免振幅过大。

### TODO 4 — AdditiveSynthesizer::appendNote

根据一个 `Note` 生成对应的一段采样。

核心流程：

1. 根据音名得到频率

3. 对每个采样点计算时间 `t`
4. 计算波形值
5. 乘以 ADSR 包络和音量
6. 限制范围到 `[-1.0, 1.0]`

## 四、编译与运行

## 六、常见坑

- **忘记把所有 `.cpp` 一起编译**：分文件编写时必须把 `src` 下所有 `.cpp` 都加入编译。
- **新增函数后只写了 `.cpp`，忘记 `.h` 声明**：例如新增 `createMySong()` 时必须在 `Song.h` 中也添加声明。
- **头文件 include 路径找不到**：编译时需要加 `-Iheader`。
- **音量太大导致爆音**：调小 `SynthConfig` 中的 `amplitude`，例如 `0.30`。
- **音符结尾有咔哒声**：增大 `release` 让音量逐渐衰减到 0。
- **修改 Makefile 后无法 clean**：当前 Makefile 已做系统判断，Windows 用 `del`，Linux 用 `rm -f`。

## 七、思考题

1. 为什么单一正弦波听起来比较单薄？

2. 谐波叠加为什么能让声音更丰富？

3. ADSR 包络中的 Release 阶段有什么作用？如果去掉 Release，声音可能会出现什么问题？

4. 如果采样率从 44100 Hz 降低到 8000 Hz，声音质量会发生什么变化？

5. 当前程序一次只能播放单个旋律线。如果要实现和弦，`Song` 或 `Note` 的数据结构应该如何修改？

6. 当前音色由 `synthSample()` 中的谐波比例决定。如果要模拟不同乐器，可以如何修改这些比例？
